expandView by Bek

What is expandView?

It's a JQuery plugin which allows you to display items with their sub items. You can easily show and hide the nested elements by clicking the arrows.

From which components each item aparts?

First you need to define the id, which is a unique string.

Then you need to define the title, which is html string.

Optionally, you can specify the total number of sub items or a loading text to be displayed until the nested items come off.

Finally, if there are any sub items, you need to implement a function which returns them as array.

How to implement expandView?

$('#expandView-demo').expandView({
items: [
{
id: 'a',
title: 'Level A',
loadingText: 'Loading...',
totalChildren: 3,
children: function() {
return [
{
id: 'a1',
title: 'Level A1',
hasChildren: false
},
{
id: 'a2',
title: 'Level A2',
children: function() {
return [
{
id: 'a22',
title: 'Level A22',
hasChildren: false
}
]
}
}
]
}
},
{
id: 'b',
title: 'Level B',
children: function() {
return [
{
id: 'b1',
title: 'Level B1',
hasChildren: false
}
]
}
}
]
});
